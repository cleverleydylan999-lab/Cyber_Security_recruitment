## 🔧 Customization Examples

### Change Brand Color
```css
/* In styles.css, change :root */
--accent: #ff00ff;  /* Change from cyan to magenta */
```
**Result**: All cyan accents update automatically

### Add New Job Position
```html
<!-- Copy job card and modify -->
<div class="job-card" data-level="mid">
    <div class="job-header">
        <h3>Your New Job Title</h3>
        <span class="level-badge mid">Mid-Level</span>
    </div>
    <!-- ... rest of card ... -->
</div>
```

### Change Company Stats
```html
<!-- In stats section -->
<div class="stat-card">
    <h3>1000+</h3>  <!-- Change number -->
    <p>Your New Stat</p>
</div>
```

### Modify Benefits
```html
<!-- In why-us section -->
<div class="benefit-card">
    <div class="benefit-icon">
        <i class="fas fa-icon-name"></i>
    </div>
    <h3>Benefit Title</h3>
    <p>Benefit description</p>
</div>
```
