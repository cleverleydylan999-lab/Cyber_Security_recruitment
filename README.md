# CyberShield Recruitment Website ✅

A professional cybersecurity recruitment website with psychological persuasion features inspired by Nike's design. Features 6 job opportunities with detailed application forms and 6 benefit pages explaining why candidates should join.

## 🎯 Features

### ✨ What's New - Benefit Pages
Each benefit card on the main page now redirects to a detailed page with in-depth information:

- **Career Growth** - Mentorship, training programs, career progression
- **Elite Team** - Team expertise, collaboration, industry recognition  
- **Innovation Hub** - Cutting-edge tools, technology stack, emerging tech
- **Competitive Pay** - Salary ranges, bonuses, benefits breakdown
- **Global Impact** - Real-world impact stories, global presence
- **Continuous Learning** - Certifications, training platforms, learning paths

### 📋 Job Opportunities (6 Positions)
Each job has its own detailed page with:
- Complete job description and responsibilities
- Required and preferred qualifications
- Role-specific benefits
- **CV Submission Form** with file upload
- Similar positions recommendations

### 🎨 Design Highlights
- Professional dark cybersecurity theme
- Cyan accent colors with hover effects
- Responsive design (mobile, tablet, desktop)
- Smooth animations and transitions
- Nike-inspired psychological elements

## 🚀 Quick Start

1. **View the Website:**
   ```bash
   # Open index.html in your browser
   # No build process needed - pure HTML/CSS/JS
   ```

2. **Explore Benefits:**
   - Click any benefit card to see detailed information
   - Each page has extensive content and testimonials

3. **Browse Jobs:**
   - Click "View Details" on any job card
   - Fill out CV application form
   - Receive success confirmation

## 🎯 Navigation Flow

```
index.html (main page)
├─ Click benefit card → Detailed benefit page (6 pages)
│  └─ Click "Explore Roles" → Back to jobs
│
└─ Click job card → Job detail page (6 pages)
   ├─ Read job description
   ├─ Fill CV form
   └─ Submit application
```
