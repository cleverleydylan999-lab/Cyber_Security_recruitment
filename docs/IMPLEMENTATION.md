
## 🔧 Customization Guide

### Change Brand (2 minutes)
1. Edit CSS variables in `styles.css`
2. Change `--primary`, `--accent`, colors
3. All components update automatically

### Add Job Position (5 minutes)
1. Copy a job card in `index.html`
2. Update title, level, salary, location, skills
3. Pick appropriate level badge (Senior/Mid/Junior)

### Modify Content (5 minutes)
1. Edit any text in `index.html`
2. Update stats, benefits, testimonials
3. Change company name "CyberShield" to your company

### Connect Backend (30 minutes)
1. Add form validation in `script.js`
2. Point form to your API endpoint
3. Handle form submission
4. Save applications to database

## 📊 Analytics Integration Ready

The website tracks but doesn't send anywhere. To integrate:

### Popular Analytics Services
- **Google Analytics** - Add GA script
- **Mixpanel** - Track custom events
- **Segment** - Central analytics hub
- **Amplitude** - Behavioral analytics
- **Custom Backend** - Send to your server

### Events Tracked
- `job_card_clicked` - User views job
- `apply_button_clicked` - User applies
- `navigation` - Page navigation
- `page_resumed` - User returns to tab
- `page_left` - User leaves tab
- `scroll_to_top_clicked` - User scrolls up
