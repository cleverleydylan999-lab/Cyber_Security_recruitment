# 🚀 Quick Start Guide

## How to View the Website

### Option 1: Direct Browser Opening (Easiest)
1. Right-click on `index.html` in VS Code
2. Select "Open with Default Browser" or "Open in External Browser"
3. The website will load in your browser

### Option 2: Using VS Code Live Server
1. Install the "Live Server" extension from VS Code marketplace
2. Right-click on `index.html`
3. Select "Open with Live Server"
4. Website opens with auto-refresh on file changes

### Option 3: Python Simple Server
Run in terminal:
```bash
cd /workspaces/Cyber_Security_recruitment
python -m http.server 8000
```
Then visit: http://localhost:8000

## 🎮 Interactive Features to Try

### 1. **Scroll Animations**
- Scroll down slowly to see cards reveal
- Watch opacity and transform animations
- Staggered timing for multiple cards

### 2. **Hover Effects**
- Hover over job cards → lift + glow
- Hover over benefit cards → subtle background animation
- Hover over skill tags → scale up

### 3. **Button Interactions**
- Click "Apply Now" button → see ripple effect → modal appears
- Click "Explore Roles" → smooth scroll to jobs section
- Click navigation links → smooth scroll transitions

### 4. **Notifications**
- Click apply button → success toast notification
- Toast appears bottom-right, auto-dismisses

### 5. **Scroll to Top**
- Scroll down past hero section
- Floating button appears bottom-right
- Click to smoothly scroll back to top

### 6. **Mouse Tracking** (Advanced)
- Move mouse over cards while scrolling
- Cards track cursor position (subtle effect)
