Job Overview

We're looking for a talented Security Engineer to design and implement robust security solutions across our infrastructure. In this entry-level role, you'll work with experienced security architects to build secure systems, implement security controls, and ensure our cloud and on-premises infrastructure meets industry security standards. You'll gain hands-on experience with modern security technologies and contribute to the development of our security posture.

Key Responsibilities

  • Design and implement security controls for cloud and on-premises infrastructure
  • Develop and maintain security policies and procedures
  • Perform security configuration assessments and remediation
  • Work with DevOps teams to integrate security into CI/CD pipelines (DevSecOps)
  • Manage identity and access management (IAM) systems
  • Implement and manage firewall, VPN, and network security solutions
  • Conduct security testing and validate security implementations
  • Monitor and respond to security alerts and events
  • Document security solutions and maintain runbooks
  • Stay current with security best practices and emerging threats

Required Qualifications

  • 1-3 years of IT operations or security operations experience
  • Strong understanding of network architecture and security fundamentals
  • Experience with cloud platforms (AWS, Azure, or GCP)
  • Knowledge of containerization (Docker, Kubernetes) and container security
  • Proficiency with Linux and Windows operating systems
  • Understanding of encryption, PKI, and authentication mechanisms
  • Familiarity with Infrastructure as Code (IaC) tools
  • CompTIA Security+ or equivalent certification

Preferred Qualifications

  • AWS Security Specialty or Azure Security Engineer certification
  • Experience with Terraform or CloudFormation
  • Knowledge of OWASP and secure coding practices
  • Scripting experience in Python or Bash
  • Familiarity with vulnerability scanning tools
